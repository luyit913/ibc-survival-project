# IBC HER2 Vaccine Survival Analysis

## Overview
This project evaluates the survival impact of a HER2-directed vaccine in inflammatory breast cancer (IBC) using an external comparison design.

Due to data access and privacy restrictions, only presentation materials are shared.

## Project Summary
- Integrated clinical cohort data with external registry-derived controls
- Constructed an analysis-ready dataset through data harmonization
- Applied propensity score matching for comparability
- Evaluated overall survival using Kaplan–Meier and Cox models
- Conducted sensitivity analyses including landmark analysis and subgroup analysis
